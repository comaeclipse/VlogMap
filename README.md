# VlogMap

An interactive platform for exploring and discovering filming locations from YouTube videos around the world. Browse where your favorite content creators have traveled, organized by country, state, and city.

## What it does

VlogMap lets you:
- **Explore locations** on an interactive map with video markers
- **Browse by geography** with hierarchical organization (USA: State → City, International: Country → City)
- **Discover creators** and their travel destinations
- **View location pages** showing all videos filmed at specific landmarks or cities
- **Watch timeline clips** with synchronized map markers for multi-location videos

## Features

- Interactive Leaflet map with clustered markers
- Location hierarchy with cities and landmarks (e.g., "Hvalsey Church" in Qaqortoq, Greenland)
- Video timeline with location-based timestamps
- Creator profiles with channel statistics
- Public browsing interface + password-protected admin panel
- Automated location matching and geocoding

## Tech Stack

Next.js 16 • React 19 • TypeScript • Tailwind CSS v4 • shadcn/ui • Leaflet • PostgreSQL
